# Homebrew

## 이게 뭐예요?
**Homebrew는 맥북 전용 "앱스토어" 같은 거예요.** 다만 보통 앱스토어처럼 마우스로 클릭해서 까는 게 아니고, 터미널에 명령어 한 줄을 쳐서 깝니다. 예를 들면 `brew install git` 이렇게요.

개발자들이 쓰는 도구들(Git, Python, Ollama 같은 거)은 대부분 앱스토어에 없어서, Homebrew로 깔아야 해요. 그래서 맥북에서 개발 환경을 만들 때는 **제일 먼저** 깔아야 하는 도구입니다.

## 왜 이걸 쓰나요?
- 도구를 깔거나, 업데이트하거나, 지울 때 명령어 하나로 해결돼요.
- 앞으로 이 가이드에서 깔 도구들(예: Ollama, Python)이 전부 Homebrew로 깔립니다.
- 맥북에는 이런 게 기본으로 없어서, 직접 깔아줘야 해요.

## 설치하는 법

1. 공식 사이트로 들어가요: https://brew.sh/
![[Xnip2026-04-25_19-37-39.jpg]]

2. 사이트 가운데에 까만 박스 안에 명령어가 보입니다. 박스 옆의 복사 아이콘을 눌러서 명령어를 복사해요.

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

![[Xnip2026-04-25_19-38-12.jpg]]

3. 아까 깔아둔 **WezTerm**을 열어요. 그리고 복사한 명령어를 붙여넣고 엔터를 칩니다.
![[Xnip2026-04-25_19-38-58.jpg]]

4. 맥북 비밀번호를 입력하라고 나와요. 비밀번호를 치고 엔터.

> ⚠️ **중요!** 터미널은 비밀번호를 칠 때 화면에 아무것도 안 보입니다. 별표(\*)도 안 나와요. 그냥 안 쳐지는 것처럼 보이지만 사실은 잘 입력되고 있는 거니까, 그냥 키보드로 타이핑하고 엔터를 누르면 돼요.

![[Xnip2026-04-25_19-39-55.jpg]]

5. 한참 진행되다 보면 맥북에서 **"명령어 도구를 설치하시겠습니까?"** 같은 팝업이 떠요. "설치"를 누릅니다. (이건 xcode-select라는 도구인데, Homebrew가 작동하려면 필요해요.)
![[Xnip2026-04-25_19-42-30.jpg]]

![[Xnip2026-04-25_19-42-42.jpg]]

## 잘 깔렸는지 확인
설치가 끝나면 **WezTerm을 한 번 껐다 켜요.** (이건 꼭 해야 해요. 안 그러면 명령어 인식이 안 됩니다.)

다시 켠 뒤에 이렇게 입력해보세요:

```sh
brew --version
```

이렇게 버전 정보가 뜨면 성공이에요! 🎉
![[Xnip2026-04-26_12-53-36.jpg]]

## 에러가 났을 때

### "맥북 버전이 낮다" 에러
설치 도중에 이런 빨간 에러 메시지가 나올 수 있어요:
![[Xnip2026-04-25_19-43-41 1.jpg]]

이건 맥북의 운영체제(macOS) 버전이 너무 낮아서 생기는 문제예요. 해결 방법은 간단합니다:

1. **사과 메뉴(🍎) → 시스템 설정 → 일반 → 소프트웨어 업데이트**로 들어가요.
![[Xnip2026-04-26_12-06-03.jpg]]

2. 업데이트 버튼을 누르고 기다립니다. **약 15분 정도** 걸리니까 그동안 다른 일을 하고 있어도 돼요.
![[Xnip2026-04-26_12-10-18.jpg]]

3. 업데이트가 끝나면 맥북이 자동으로 다시 켜져요.
![[Xnip2026-04-26_13-04-43.jpg]]

4. 다시 WezTerm을 열고, 위에서 복사한 Homebrew 설치 명령어를 한 번 더 붙여넣어요. 이번엔 잘 진행될 겁니다.
![[Xnip2026-04-26_12-47-24 1.jpg]]

## 알아두면 좋은 명령어
설치를 마치면 이런 명령어들을 자주 쓰게 될 거예요:

| 명령어 | 무슨 뜻? |
|---|---|
| `brew install 이름` | 도구를 새로 깔아요 |
| `brew update` | Homebrew 자체를 최신으로 만들어요 |
| `brew upgrade` | 깔아둔 도구들을 최신으로 만들어요 |
| `brew list` | 내가 깔아둔 도구들의 목록을 보여줍니다 |
| `brew uninstall 이름` | 도구를 지워요 |
