# llama.cpp

## 이게 뭐예요?
**llama.cpp는 ChatGPT 같은 AI 모델을 내 맥북에서 직접 돌리게 해주는 프로그램이에요.** 보통 ChatGPT는 OpenAI 서버에서 돌아가서 인터넷이 필요하지만, llama.cpp를 깔면 모델 파일 하나만 다운받아서 인터넷 없이 내 컴퓨터에서 그대로 돌릴 수 있어요. 이런 도구를 "추론 엔진(inference engine)"이라고 부릅니다.

## 왜 이걸 쓰나요?
- **인터넷이 없어도 돼요** — 비행기 안에서든, 와이파이가 끊겼든 그대로 작동합니다.
- **데이터가 외부로 안 나가요** — 내가 입력한 모든 글이 내 맥북 안에서만 처리됩니다. 회사 기밀이나 개인적인 글을 다뤄도 안전해요.
- **공짜예요** — ChatGPT Plus 같은 구독료를 안 내도 됩니다.
- **새 모델 지원이 빠릅니다** — 새로운 AI 모델이 나오면 보통 같은 날 llama.cpp용 파일(GGUF 형식)이 같이 풀려요.
- **속도가 빠릅니다** — 일상적인 작업(긴 문서 요약, 여러 번 주고받는 대화)에선 MLX나 Ollama보다 더 빠른 경우가 많습니다.
- **Apple Silicon에 최적화** — M4 Max 같은 맥북 칩의 GPU(Metal)와 통합 메모리를 잘 활용해요.
- **단점** — 명령어로 다루는 도구라 처음엔 좀 낯설 수 있어요. 대신 기본 명령어 몇 개만 외우면 됩니다.

## 설치하는 법

1. **WezTerm**을 열어요. (1편에서 깔아둔 그 터미널이에요.)

2. 아래 명령어를 입력하고 엔터:

```sh
brew install llama.cpp
```

3. Homebrew가 알아서 필요한 것들을 다운받고 설치합니다. 1~2분 정도 걸려요.
![[Xnip2026-04-26_15-02-26.png]]
## 잘 깔렸는지 확인

설치가 끝나면 아래 명령어로 확인해보세요:

```sh
llama-cli --version
```

이렇게 버전 정보가 뜨면 성공이에요! 🎉
